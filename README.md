# QuranBro

A modern and minimalist web application for reading Al-Quran, designed for ease of use and a premium reading experience.

## Features

- Full 114 Surahs with translation and transliteration.
- Juz navigation (30 Juz).
- Customizable reading experience:
  - Six high-quality Arabic fonts (Amiri, Scheherazade New, Lateef, etc.).
  - Adjustable font sizes for Arabic, Latin, and Translation text.
  - Global theme support (Light, Dark, and System).
- Premium modern UI with app-like animations and bottom sheet navigation.
- Session persistence (automatically saves your settings).
- History tracking for recently read verses.

## Tech Stack

- React 18
- Vite 7
- Tailwind CSS 4
- Lucide React (Icons)
- React Router 7
- React Helmet Async (SEO)

## Getting Started

### Prerequisites

- Node.js (version 18 or later)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rmdnv/quran.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Scripts

- `npm run dev`: Start the development server.
- `npm run build`: Build the application for production.
- `npm run preview`: Locally preview the production build.
- `npm run lint`: Run ESLint to check for code issues.
